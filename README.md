# Scene_Bosque_Compartir
Mini escena creada en Unity (URP) para demostrar ensamblaje de assets, materiales PBR e iluminación.

Incluye materiales PBR ajustados, iluminación ambiental, efectos de postprocesado y un ensamblaje limpio de assets.

---

## Características principales

- Implementación de URP para mejorar rendimiento y calidad visual.  
- Materiales PBR corregidos y convertidos desde shaders estándar.  
- Post-Effects aplicados (Bloom).  
- Organización de escena y jerarquía optimizada.  
- Integración de assets y terrenos naturales para una composición visual equilibrada.

---

## Objetivo del proyecto

Demostrar dominio básico-intermedio en:
- Integración y ensamblaje de assets.
- Ajuste visual de materiales e iluminación.
- Comprensión del flujo de trabajo en URP.
- Aplicación de criterios estéticos y técnicos.
